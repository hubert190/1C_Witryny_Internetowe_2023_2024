Strict DTD
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
	"http://www.w3.org/TR/html4/strict.dtd">

Transitional DTD
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
	"http://www.w3.org/TR/html4/loose.dtd">

Frameset DTD
	<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN"
	"http://www.w3.org/TR/html4/frameset.dtd">

Strict (ścisła): Ta wersja narzuca najbardziej rygorystyczne standardy związane z poprawnością struktury dokumentu. Ogranicza użycie elementów i atrybutów do tych, które są zgodne z normami HTML 4.

Transitional (przejściowa): Jest bardziej elastyczna niż ścisła wersja i pozwala na korzystanie z elementów i atrybutów, które są przestarzałe lub niezalecane w nowszych wersjach HTML.

Frameset: Ta wersja jest przeznaczona do dokumentów zawierających ramki (frames). Jest stosowana, gdy strona internetowa używa konstrukcji z ramkami.
